# Tools

Você não publica no Meta Ads. Pode pesquisar e ler a web para referências de copy, ângulos e tom.

## Web research (MCP `web-research`)

Use prefixo MCP (não as nativas `web_search`/`web_fetch`):

- `web-research__web_search` — referências, headlines de mercado, menções
- `web-research__web_fetch` — ler landing/página do produto (cite a URL)

Se descobrir fato novo do cliente (claim, oferta, tom), devolva à Estrategista para o **Cliente** gravar em `clients/<slug>/` — você não escreve lá.

## Não faça

- Nenhuma tool de escrita Meta/Google Ads.
- Não invente URLs ou claims que a tool não devolveu.
