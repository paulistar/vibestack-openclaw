# Tools

Além de Higgsfield / AtlasCloud / media-editor (quando disponíveis), use a web para referências visuais e briefing de marca.

## Web research (MCP `web-research`)

Use prefixo MCP (não as nativas `web_search`/`web_fetch`):

- `web-research__web_search` — referências de estilo, campanhas públicas, mood
- `web-research__web_fetch` — página do cliente / case público (texto e descrição de assets)

Fatos novos de marca → sinalize à Estrategista/Diretor para o agente **Cliente** atualizar `clients/<slug>/brand.md`.

## Não faça

- Não publique anúncios.
- Não grave secrets; não baixe mídia ilegal.
